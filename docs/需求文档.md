# docs/需求文档.md
日期：2025-06-18
需求提出人：用户

## 功能需求
为Hugo博客添加一个统计页面，显示实时评论数量和文章总数量

## 具体要求
1. 创建一个新的页面，展示博客的统计数据
2. 显示文章总数量（自动统计content/posts目录下的文章）
3. 显示实时评论数量（通过giscus API获取GitHub Discussions中的评论数）
4. 页面设计美观，与博客主题风格一致
5. 数据能够实时更新

## 验收标准
1. 页面能够正确显示文章总数量
2. 页面能够正确显示实时评论数量
3. 页面设计符合博客主题风格
4. 数据能够实时更新
5. 页面在博客导航栏中可访问
