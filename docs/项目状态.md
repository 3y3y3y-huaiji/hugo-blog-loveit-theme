# 项目状态

更新日期：2026-06-18

## 项目概述

本项目是一个全自动运行的 AI 科技博客系统。通过 GitHub Actions 触发定时任务，抓取最新科技 RSS 资讯热点，随机从 5 大模型池中挑选模型调用 NVIDIA NIM API 撰写科技前沿博文，并自动部署到 Cloudflare Workers Assets。

|------|------|
